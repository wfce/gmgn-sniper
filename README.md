# GMGN首发代币狙击器

用于 gmgn.ai 的 Chrome 扩展：在代币列表中标记同名代币的「首发 / 非首发」，为非首发提供一键跳转首发按钮，并提供“自动狙击购买（Auto Snipe / Auto Buy）”能力：当短时间内出现多个同名代币时，自动识别首发并尝试点击 GMGN 内置 Buy/QuickBuy 按钮完成买入（仅模拟页面点击，不是链上机器人）。

## 功能简介

A. 首发识别与跳转（基础能力）
- 启用/禁用开关：一键关闭扩展逻辑，恢复原始页面
- 首发/非首发自动标记：
  - 将同名代币分组（由“匹配模式”决定：Symbol / Name / 任一 / 同时）
  - 在时间窗口内比较，同组中“最早创建/最老”的代币视为首发，其余为非首发
- 非首发一键跳转首发：
  - 非首发代币旁显示“打开首发/Open First”
  - 点击在新标签页打开对应首发代币详情页
- 时间窗口：可设置比较范围（默认 120 分钟）
- 双语言：中文 / English（默认跟随浏览器语言，可手动切换）
- 显示模式：目前仅支持“显示所有（仅标记）”，其余显示为“开发中”

B. 自动狙击购买（重点功能）
- 自动狙击（Auto Snipe / Auto Buy）：
  - 只监控第一列（New Created / 新创建）
  - 在 Y 秒内出现 X 个同名代币时触发
  - 自动选出这组同名代币里的“首发”
  - 尝试定位该首发行并点击 GMGN 内置 Buy/QuickBuy 按钮完成买入
- 防重复与锁定：
  - 同一个 token（chain:address）只自动买一次
  - 同一个同名组触发后会被标记为已触发，避免重复触发连续下单
- 统计与提示：
  - 记录 Detections（触发检测次数）、Auto Buys（自动买入次数）、Today（今日买入）
  - 自动买入成功会弹出页面提示

重要提示（自动狙击购买）
- 本扩展不是链上交易机器人：仅自动操作 gmgn.ai 页面上的买入按钮。
- 请务必先在 GMGN 页面里设置好默认买入金额/QuickBuy 参数，否则触发后也可能买不成功。
- gmgn.ai 的 UI/按钮结构变化、弹窗确认、风控提示、网络延迟等，都可能导致自动点击失败或与预期不一致。请谨慎启用。

## 安装方式

A. 推荐：Chrome 网上应用商店安装
1. 打开商店链接
2. 点击“添加至 Chrome”
3. 安装完成后访问 gmgn.ai 即可生效

B. 备选：GitHub 手动安装（开发者模式）

方式 A：git clone
1) 克隆仓库
   git clone https://github.com/wfce/gmgn-filter.git
2) 进入目录（确认包含 manifest.json）
   cd gmgn-filter

方式 B：下载 ZIP
1) Code → Download ZIP
2) 解压到本地目录
3) 确保扩展根目录包含 manifest.json

在 Chrome 中加载
1) 打开 chrome://extensions/
2) 开启右上角 Developer mode（开发者模式）
3) 点击 Load unpacked（加载已解压的扩展程序）
4) 选择包含 manifest.json 的扩展根目录
5) 加载成功后刷新 gmgn.ai 页面

提示：若提示找不到 manifest.json，说明选错了目录层级。

## 如何使用

1) 打开 gmgn.ai
https://gmgn.ai/

2) 点击扩展图标打开设置面板
- 确保“启用”开关为开启
- 设置语言、时间窗口、匹配模式

3) 查看列表标记
- 首发：显示“首发/First”标记（视觉高亮）
- 非首发：显示“非首发/Not First”标记 + “打开首发/Open First”按钮

4) 使用一键跳转
- 点击“打开首发/Open First”直接打开对应首发 token 页面

5) 启用自动狙击购买（可选，高风险）
- 开启 Enable Auto Snipe
- 设置：
  - Time Window (sec)：Y 秒
  - Min Duplicates：X 个
- 确保 GMGN 内部 QuickBuy 已设置好买入金额
- 当第一列出现满足条件的同名代币时，扩展将尝试自动买入首发

## 设置说明

基础设置
- 启用：总开关
- 语言：Auto / 中文 / English
- 时间窗口（分钟）：用于首发判定比较范围
- 匹配模式（Match Mode）：
  - Symbol only：仅 Symbol 相同视为同名
  - Name only：仅 Name 相同视为同名
  - Symbol or Name：任一相同视为同名
  - Both Symbol & Name：两者都相同才视为同名
- 仅在时间窗口内比较：开启可减少旧代币干扰
- 显示模式：目前锁定为“显示全部（仅标记）”

自动狙击（Auto Snipe）
- Enable Auto Snipe：开启后才会自动尝试买入
- Time Window (sec)：触发判定窗口（秒）
- Min Duplicates：触发所需最少同名数量
- 仅监控第一列：只对 New Created / 新创建列有效
