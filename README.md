# GMGN 同名首发过滤器（GMGN Same-Name First Launch Filter）

用于 gmgn.ai 的 Chrome 扩展：在代币列表中标记同名代币的「首发 / 非首发」，并在非首发代币旁提供一键跳转到对应首发代币页面的按钮。

## Chrome 商店状态：审核中（In Review）

本扩展目前正在提交 Chrome Web Store 审核中，暂未正式上架。  
在上架前，你可以通过本仓库源码手动安装使用（下方有详细教程）。

---

## 当前支持的链

目前支持在 gmgn.ai 上识别并跳转以下链的代币页面：
- BSC
- SOL
- ETH

---

## 功能简介

- **启用/禁用开关**：可一键关闭过滤功能，保留原始页面显示
- **首发/非首发标记**：默认按「Symbol 或 Name 相同」分组，在时间窗口内比较，同组内 age 最大（最老）的代币视为「首发」
- **非首发一键跳转**：在「非首发」标签旁显示「打开首发」按钮，点击直接打开对应首发代币 token 页面（新标签页）
- **时间窗口**：可设置比较的时间范围（默认 120 分钟）
- **双语言**：中文 / English（默认跟随浏览器语言，可手动切换）
- **显示模式**：当前仅支持「显示所有（标记首发/非首发）」；其他模式显示为「开发中」

## 安装（GitHub 手动安装）

由于扩展仍在 Chrome 商店审核中，请按以下步骤手动安装。

### 方式 A：使用 git clone

1. 克隆仓库到本地  
   ```bash
   git clone https://github.com/wfce/gmgn-filter.git
   ```

2. 进入项目目录（确认这里包含 manifest.json）  
   ```bash
   cd gmgn-filter-main
   ```

### 方式 B：下载 ZIP

1. 打开本仓库页面，点击右上角 **Code → Download ZIP**  
2. 解压到本地任意目录  
3. 确保解压后的扩展根目录内包含 `manifest.json`

---

## 在 Chrome 中加载扩展（开发者模式）

1. 打开 Chrome，进入：  
   `chrome://extensions/`

2. 打开右上角：  
   **Developer mode（开发者模式）**

3. 点击：  
   **Load unpacked（加载已解压的扩展程序）**

4. 选择扩展根目录（必须是包含 `manifest.json` 的那一层目录）

5. 加载成功后，扩展会出现在扩展列表中

> 提示：如果你选错目录（例如选到了外层或内层），Chrome 会提示找不到 manifest.json。请重新选择正确目录。

## 如何使用

1. **打开 gmgn.ai**：  
   https://gmgn.ai/

2. **点击扩展图标打开设置**：
   - 确保「启用」开关已打开
   - 根据需要调整语言、时间窗口、匹配模式等设置
   - 点击「保存」应用设置

3. **在代币列表/表格中查看标记**：  
   - **首发**：绿色描边 + 左侧绿色竖条 + "首发/First" 标签
   - **非首发**：灰色遮罩 + "非首发/Not First" 标签 + "打开首发/Open First" 按钮

4. **点击非首发行的「打开首发」按钮**：  
   - 直接打开对应首发代币的 token 页面（新标签页）

5. **临时关闭过滤功能**：
   - 点击扩展图标，关闭「启用」开关即可

## 设置说明

点击浏览器工具栏中的扩展图标，即可打开设置面板。

### 可配置项

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用 | 是否启用过滤功能 | 开启 |
| 语言 | 界面语言（自动/中文/English） | 自动（跟随浏览器） |
| 时间窗口 | 比较的时间范围（分钟） | 120 |
| 匹配模式 | Symbol/Name 的匹配方式 | Symbol 或 Name 相同 |
| 仅在时间窗口内比较 | 是否仅比较时间窗口内的代币 | 开启 |
| 显示模式 | 显示全部/仅首发/仅非首发（开发中） | 显示全部 |

> 注意：显示模式目前锁定为「显示所有（标记首发/非首发）」，其他模式为开发中不可切换。

| 权限 | 用途 |
|------|------|
| `storage` | 用于保存扩展设置（启用状态、语言、时间窗口、匹配模式等） |
